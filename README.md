# Mobile-Programming-HWs Profile

This repo controls the public GitHub organization profile for Mobile Programming HWs.

The visible organization profile is:

```text
profile/README.md
```

## What The Profile Shows

- Links to each public org repo.
- Short explanation for each project.
- Screenshots pulled from each project repo.
- Shared setup notes for Android and Swift projects.
- Notes about where screenshots and implementation details live.

## Suggested Repo Descriptions

Use these in the GitHub repository description field:

| Repo | Description |
| --- | --- |
| `mobile-programming-hws.github.io` | GitHub Pages project hub for the Mobile Programming HWs apps, screenshots, and source links. |
| `HW1` | Android course browser and weekly schedule planner for Sharif course data. |
| `HW2` | Android trivia game with local users, settings, scores, and Open Trivia DB questions. |
| `Project` | Android course and homework management app with teacher, TA, and student roles. |
| `Swift` | Swift cryptocurrency tracker with terminal, SwiftUI, sample data, and tests. |
| `.github` | Public GitHub organization profile for the Mobile Programming homework projects. |

## Editing Notes

- Keep the profile short enough to scan on the organization page.
- Use raw GitHub image links for screenshots from other repos.
- Update the profile after app READMEs or screenshot paths change.
- Keep setup commands aligned with the app repos.
